# Bem-vindo ao FrellasOrc

Este é um app para gerar orçamentos de projetos para freelancers. Com ele, você pode criar orçamentos detalhados de forma rápida e organizada, facilitando a comunicação com clientes e agilizando o fechamento de projetos.

### O app permite

Gerar orçamentos personalizados para diferentes tipos de projetos.
Organizar informações do projeto de forma prática.
Salvar, exportar e compartilhar orçamentos rapidamente.
Otimizar o tempo do freelancer, transformando ideias em propostas profissionais em poucos cliques.
